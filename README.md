# Width: Who Can Take Over The Screen.
A two-player game in which you must defeat your opponents by clicking to take over the screen.

## Keyboard shortcuts
- Player One: F
- Player Two: J
- New Game: Space
- Volume on: Up arrow
- Volume off: Down arrow
